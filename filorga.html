Filorga NCEF-Revitalize Serum

Personal exploration around Filorga’s NCEF Revitalize Serum, covering art direction, 3D modeling, texturing, lighting, and rendering.

The project was conceived as an experimentation with fluid simulations and maximalist aesthetics, using dynamic compositions, rich materials, and layered visual elements to amplify the product’s identity.

The goal was to push both the technical and visual aspects of the render, balancing realism with a more expressive and elevated artistic direction.

This project is part of an ongoing independent R&D process.
