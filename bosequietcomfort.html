Bose
QuietComfort

Developed as part of an ongoing research process, this project is a focused visual study that explores the interaction between light, texture and form on the Bose QuietComfort headphones. This demonstrates how lighting and material decisions can be used as strategic tools to reinforce quality, performance and tactility.

This project is part of an ongoing independent R&D process.

Behind the scenes

Anisotropic surfaces were intentionally applied to the headphones to create nuanced lighting reactions, allowing light to travel, break and accentuate the product’s geometry.
