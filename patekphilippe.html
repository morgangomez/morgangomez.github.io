Patek Philippe
Nautilus 5990/1A-011

Inspired by the Nautilus’ maritime heritage and its distinctive porthole silhouette, this project translates the watch’s identity into a dynamic visual environment.

Flowing water elements were developed using Blender’s Geometry Nodes, enabling fully procedural, animated streams that interact organically with the watch.

Particular attention was given to lighting and material definition, with each scene carefully constructed using multiple light sources to reveal the precision, depth and craftsmanship of the timepiece.

This project is part of an ongoing independent R&D process.
